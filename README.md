# fastapi

## Get Started
1. Clone the repository
```bash
git clone git@github.com:eclipsegst/flask.git
```

2. Install dependencies
```bash
python -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

3. Run the app
```bash
uvicorn main:app --reload
```
4. Check it
- http://127.0.0.1:8000
- API docs: 
  - http://127.0.0.1:8000/docs
  - http://127.0.0.1:8000/redoc
